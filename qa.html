QA Automation Framework

Development Language

Java 1.6

Technology Stack

  • Spring

Description

In this project we created a Selenium based automation tool to dramatically increase the speed and range of the tests suites the QA engineers handled. By representing our web application as a set of page objects and integrating them with Selenium core we were able to produce a framework which we could easily configure to fit any scenario
